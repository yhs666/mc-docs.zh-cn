尝试创建新的 Azure 虚拟机 (VM) 时，遇到的常见错误是预配失败或分配失败。

* 当由于准备步骤不当，或者在从门户捕获映像期间选择了错误的设置而导致 OS 映像无法加载时，将发生预配失败。
* 当群集或区域没有可用的资源或无法支持所请求的 VM 大小时，会发生分配失败。
