## <a name="data-box-heavy-limits"></a>Data Box Heavy 限制

- Data Box Heavy 每个节点最多可存储 10 亿个文件。
- Data Box Heavy 支持云中每个节点最多 512 个容器或共享。 用户共享中的顶层目录将成为云中的容器或 Azure 文件共享。
